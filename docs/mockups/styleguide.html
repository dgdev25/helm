Glass Surfaces

Three glass surface variants. All use backdrop-filter: blur(16px).

Default glass
Project cards, panels, modals. Background: rgba(255,255,255,0.04)
Teal glass
Active nav items, primary accents. Background: rgba(34,153,113,0.15)
Accent glass
Secondary highlights, charts. Background: rgba(206,220,0,0.08)

Gradients

Primary gradient runs teal→indigo (135°). Surface gradient fades from teal-tinted to transparent.

Motion System

Two speeds. Hover states: 200ms. State changes and cards: 300–400ms. Entrance animation: fadeInUp 0.4s.

--transition-fast: 200ms ease
--transition-normal: 300ms ease
--animation-entrance: fadeInUp 0.4s ease both
stagger delay: nth-child × 0.06s
